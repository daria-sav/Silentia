# Silentia

**EN**  
Silentia is a 2D puzzle-platformer with a clone-based replay mechanic.  
The player guides an old monk through a quiet temple and uses recorded echoes of chakra characters to solve puzzles across time.

**ET**  
Silentia on 2D mõistatus-platvormer kloonipõhise taasesitusmehaanikaga.  
Mängija juhib vana munka läbi vaikse templi ning kasutab tšakrate salvestatud kajasid, et lahendada ajastusel ja koostööl põhinevaid mõistatusi.

## Gameplay / Mänguprotsess

**EN**  
The core gameplay is based on recording and replaying actions.  
The player can record the movement of different chakra characters and later use these recordings as echoes. Echoes can activate buttons, move platforms, reach inaccessible areas, and help the monk complete the level.

**ET**  
Mängu põhimehaanika tugineb tegevuste salvestamisele ja taasesitamisele.  
Mängija saab salvestada erinevate tšakrate liikumist ning kasutada neid hiljem kajadena. Kajad saavad vajutada nuppe, liigutada platvorme, jõuda mungale kättesaamatutesse kohtadesse ja aidata taset läbida.

## Features / Omadused

- 2D puzzle-platformer gameplay  
- Clone-based recording and replay system  
- Three playable chakra characters  
- Puzzles based on timing, planning, and coordination  
- Pixel-art visual style  
- Developed in Unity with C#
